🔍 WHY USE THIS?

When you receive a suspicious email, the visible "From" name can be faked. Email headers contain the technical fingerprints that reveal the true origin — the real sending IP, whether the domain is authenticated, and if the message was tampered with in transit. This is the same process IT security professionals use when triaging phishing reports.

📋 HOW TO GET HEADERS

Gmail: Open email → ⋮ → "Show original"
Outlook: Open email → ⋮ → "View message source"
Apple Mail: View → Message → All Headers
Yahoo: Open email → ⋮ → "View raw message"

🛡️ WHAT WE CHECK

SPF — Confirms the sending server is authorized by the domain owner. DKIM — Verifies the email wasn't altered after it was sent. DMARC — Ensures SPF and DKIM align with the "From" domain. Together, these three protocols are the industry standard for detecting spoofed and phishing emails in enterprise IT environments.
